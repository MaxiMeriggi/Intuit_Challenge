# Intuit_Challenge

Solución al challenge técnico de Intuit, desarrollada con .NET 6 siguiendo una arquitectura limpia (API, Application, Domain, Infrastructure).

# Requisitos

- .NET 6 SDK

- Visual Studio 2022 o VS Code

- MySql

# Instalación

Clonar el repositorio:

> git clone https://github.com/MaxiMeriggi/Intuit_Challenge.git


Restaurar paquetes:

> dotnet restore

Configurar las variables necesarias en:

> Intuit.Api/appsettings.Development.json

Abrir la solución y ejecutar el proyecto Intuit.Api.

Swagger disponible en:

> https://localhost:{puerto}/index.html
